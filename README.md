## You don't have a copy or project??

Frontend Setup:

    $ git clone https://github.com/Ugyenwangdi/Druk-eBird.git

    $ cd Druk-eBird

    $ cd client 

    $ npm install 

    $ npm start


Backend Setup:

    $ cd server 

    $ npm install 

    $ npm start
